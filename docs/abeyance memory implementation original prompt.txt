1. Snap Engine (Insight Activation Engine)

Purpose:

Detect when new evidence activates dormant fragments.

Expected file:

backend/app/services/abeyance/snap_engine.py

Responsibilities:
	•	compare incoming fragment embeddings against dormant fragments
	•	evaluate snap activation thresholds
	•	detect multi-fragment activations
	•	emit events via Redis Streams

Event structure:

SnapEvent {
    snap_id
    evidence_fragment_id
    matched_fragment_ids
    confidence_score
    timestamp
}
